Profil Pengembang

Pengembangan website umkm baju muslim/muslimah ini bertujuan untuk menyediakan platform yang efisien, interaktif, dan mudah diakses. Proyek ini dilaksanakan sebagai bagian dari tugas UTS dalam mata kuliah Pemrograman Web. Dalam pengembangannya, kami menggunakan berbagai teknologi web terkini seperti HTML dan CSS. Dengan adanya website ini, kami berharap dapat memberikan kontribusi dalam meningkatkan pelayanan UMKM baju muslim/muslimah, mempercepat proses administrasi, serta memudahkan masyarakat dalam mendapatkan informasi promo dan mudah diakses kapan saja dan di mana saja.
